# RoboHand
# Управление роборукой
Управление роборукой реализовано через веб-интерфейс.
Команды с сайта передаются через Flask-сервер на Python, далее по Serial-интерфейсу через USB поступают в Arduino, который управляет сервоприводом и обрабатывает данные с датчика тока.
# Что нужно для проекта:
## Программы
Arduino IDE
Python + библиотеки Python(pip install flask pyserial)
VS Code
## Компоненты для руки
Arduino Nano                  
Сервопривод
Датчик тока          
Пружины                       
Нити                  
Провода             
Макетная плата   
USB кабель                    
# Шаги:
1) Установить программы
2) загрузить основной код Arduino
3) создать Python файл и загрузить код
4) создать сайт (создать html файл) для управления и загрузить код
5) Открыть: http://127.0.0.1:5000/
